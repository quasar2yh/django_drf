# spartamarket v.0.1 (with DRF)
---------
- 기여자 : 이윤후
- 기간 : 24.05.01~02
- 목표 : DRF를 이용해서 회원가입, 로그인, CRUD를 할 수 있는 API 생성
- 사용한 서드파티 앱 :
   - django_extensions, django_seed, drf_spectacular
   - requirements.txt 참고
